# Documentation

Organized documentation for Tally Agentic Wallet.

## Structure

| Folder | Purpose |
|--------|---------|
| `investigations/` | Research, analysis, and deep-dives |
| `implementation/` | Code changes, feature specs, and implementation notes |
| `testing/` | Test results, validation reports |
| `exploration/` | Codebase exploration and architecture analysis |
| `decisions/` | Technical decisions and rationale (ADRs) |
| `reference/` | Standards, guides, and reference material |
